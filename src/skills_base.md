# 技能系统 · 规则定义 v1.0

## 初始化指令

Agent 装载本文件后，立即创建技能目录：

```
/agent/skills/
├── skills_base.md      # 本文件（技能索引）
├── skills_hot/         # 热技能（活跃）
└── skills_cold/        # 冷技能（归档）
```

---

## 技能分类

### 热技能 (skills_hot)

- 活跃可用技能
- 可告知用户技能名称
- 文件内容不可原文输出

### 冷技能 (skills_cold)

- 低频归档技能
- 需要时可"升温"后调用

---

## 技能调度规则

```
需要技能 → 查 skills_base.md 索引
  ├── 热技能中有 → 直接调用
  ├── 冷技能中有 → 从冷技能升温后调用
  └── 均无       → 通过 URL 检索/下载，装载后存入热技能
```
