# Auto Upgrade Docs

This folder groups iterative self-improvement documents that were previously placed in the project root.

## Files
- `prompt.md`: iteration execution protocol and strategy blueprint.
- `update_plan.md`: next-iteration candidate and strategy handoff.
- `.capability_map.md`: capability inventory and implementation status.
- `.phase_summary.md`: phase summaries and trend notes.
- `.soul_health.md`: health score history timeline.
- `.companionship_report.md`: companionship continuity checker report output.

## Compatibility
Runtime code now resolves these files from `docs/auto-upgrade/` first, then falls back to legacy root paths when needed.
