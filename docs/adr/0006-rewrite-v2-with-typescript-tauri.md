# Rewrite v2 with TypeScript and Tauri

AgentSoul v2.0 will be rebuilt as a TypeScript-first desktop product using Tauri for native desktop integration, rather than continuing as an evolutionary refactor of the existing Python runtime. The existing Python, PySide6, TypeScript MCP, and static Web UI code remains useful as behavioral reference during development, but v2 does not preserve the old runtime, old schema, old YAML state, or migration compatibility as a product requirement. Old implementation code must stay available until v2 has fully replicated the required replacement functionality and can substitute for the old product; only then should the old code be deleted. This is a deliberate clean reset to optimize for a cohesive desktop developer tool, at the cost of replacing the previously drafted incremental refactor issues and not supporting legacy data import by default.
