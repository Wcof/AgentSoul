# Local-first Companion

AgentSoul v2.0 is a Local-first Companion: core Companion state, Gateway routing, approval flow, credentials, audit records, session discovery, and control-center configuration run locally and do not require cloud services. This keeps prompts, code, credentials, and local work history under the user's control and supports low-latency desktop interaction. Data portability is handled first through User-managed Export; any Remote Sync capability must be explicit opt-in and must not become a prerequisite for the core experience.
