## 本次踩坑警告

- TypeScript 中 `fs.accessSync` 返回 void，不能用于 if 条件判断，必须用 try-catch
- 表达式索引类型需要显式类型断言，TypeScript 不允许 string 直接索引自定义类型
- 接口属性命名必须保持一致性（snake_case），否则 schema 验证和接口实现不匹配会导致编译错误
- 修改 TypeScript 代码后必须重新 `npm run build`，否则修改不会生效
- 正则表达式字面量必须正确加括号，`.match/` 会语法错误，必须是 `.match(/.../)`
- macOS date 命令参数格式与 GNU date 不同，脚本需要适配
- 对 Optional 类型要显式断言或者用保护语句，不能直接访问属性
- 运行 Python 模块时需要设置正确的 PYTHONPATH，否则相对导入会失败
- 纯静态 HTML 添加编辑功能只需要用 textarea 替换内容，无需复杂框架，保持简单即可
- shields.io 徽章可以直接使用静态 URL，不需要动态生成，简单有效
- 增量式扩展现有模块比创建新模块更容易保持架构一致性，测试通过率更容易维持 100%
- 为纯静态 Web UI 添加功能保持纯 HTML/CSS/JavaScript 实现不引入框架，更容易维护
- 文件读取操作始终需要异常处理，即使文件存在检查通过，实际读取仍可能因权限/磁盘问题失败，返回 None 比抛出异常更友好让上层处理
- 同名变量在同一作用域重复使用不同类型会导致 mypy 类型推断错误，应该使用不同变量名避免冲突
