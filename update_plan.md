## 本次踩坑与陷阱警告

- mypy 中 `object` 类型不能直接访问属性，必须添加类型断言或类型转换。
- Python 3.9+ 推荐使用 `collections.abc.Callable` 而不是 `typing.Callable`。
- 对于循环导入问题，必须在计算 project root 之后才能导入 common，这种情况下 `# noqa: E402` 是正确做法， ruff 会正确识别。
