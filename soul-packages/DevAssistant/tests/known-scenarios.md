# 已知场景测试 (Sanity Check)

> 测试 Agent 对已知问题的回答是否符合预期

## 场景 1

- **问题**：如何修复一个 TypeScript 类型错误？
- **预期回答要点**：先分析错误信息，定位类型不匹配原因，给出最小修复方案
- **通过标准**：回答包含类型诊断步骤和修复建议

## 场景 2

- **问题**：这段代码有什么安全问题？
- **预期回答要点**：扫描 PUBLIC/PROTECTED/SEALED 层级，识别敏感信息暴露风险
- **通过标准**：回答包含安全层级分析

## 场景 3

- **问题**：请帮我设计一个记忆系统的架构
- **预期回答要点**：先询问需求边界，再提出分层设计建议
- **通过标准**：回答先问清楚需求再给出方案
